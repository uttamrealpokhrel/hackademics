# InfiniteQuiz Generator - Hackademics

🎯 **Create unlimited multiple-choice quizzes on any topic with AI-powered question generation!**

## 📋 Project Overview

InfiniteQuiz is an innovative educational tool that leverages AI to generate dynamic 10-question multiple-choice quizzes on any topic. Featuring gamification elements like countdown timers, progress tracking, and instant feedback, it makes learning engaging and fun.

### 🎯 Key Features

- ✅ **AI-Powered Quiz Generation**: Uses OpenAI API with structured JSON output for reliable question generation
- ✅ **10-Question Quizzes**: Each quiz contains 10 carefully crafted multiple-choice questions
- ✅ **Countdown Timer**: 30-second timer per question with visual indicators
- ✅ **Progress Tracking**: Real-time score display and progress bar
- ✅ **Instant Feedback**: Detailed explanations for correct and incorrect answers
- ✅ **Multiple Topics**: Pre-loaded with 10+ popular topics
- ✅ **Custom Topics**: Users can enter any topic to generate quizzes
- ✅ **Results Download**: Export quiz results as CSV or JSON
- ✅ **Responsive Design**: Works on desktop, tablet, and mobile
- ✅ **Navigation Tools**: Home, About, Contact, Privacy, Feedback, Sign In/Up

## 🚀 Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Select a topic or create a custom one
4. Complete the 10-question quiz
5. View results and download
